# Análisis de Datos de Vehículos - Aplicación Web

## Descripción del Proyecto
Esta aplicación web interactiva permite explorar y analizar un conjunto de datos de anuncios de venta de vehículos. Desarrollada con Streamlit, proporciona herramientas visuales para entender mejor el mercado de automóviles usados.

## Para qué sirve la aplicación
La aplicación está diseñada para ayudar a usuarios interesados en el mercado automotriz a:
- Explorar tendencias de precios de vehículos
- Analizar la relación entre características del vehículo y su precio
- Visualizar la distribución de diferentes tipos de vehículos por marca

## Funcionalidades que proporciona
- **Información del dataset**: Muestra estadísticas básicas y muestra de los datos
- **Histograma interactivo**: Visualiza la distribución de precios de vehículos
- **Gráfico de dispersión**: Analiza la relación entre kilometraje y precio
- **Interfaz intuitiva**: Botones interactivos para generar visualizaciones bajo demanda

## Tecnologías utilizadas
- Python
- Streamlit
- Pandas
- Plotly Express

## Cómo ejecutar la aplicación
```bash
streamlit run app.py
```
